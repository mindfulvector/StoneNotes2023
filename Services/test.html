This is a test of a static resource.

These extensions are treated as static files:

  • .js
  • .html
  • .css
  • .jpg
  • .png
  • .gif

Not really CGI!

Files other than server.ts with the .ts extension will be executed as CGI-like scripts:

  • Request information will come in on STDIN as a JSON blob.
  • You should send your reply as HTML or JSON on STDOUT.
  • Any error in STDERR will cause a generic error to be returned, the error will be logged.
